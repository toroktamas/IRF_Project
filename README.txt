Tanuló neptun kódja: H2UL99
A projekt terv olyan alkalmazás készítése amely segítségével egy xml fájltból lehet törölni bizonyos számunkra nem releváns adatokat.
Ez a projektet azért gondoltam hasznosnak mivel hogyha rendelkezünk egy Budapesti lakásokat tartalmazó xml állománnyal mint
ahogy a példában én. Ki tudjuk szűrni belőle a számunkra megfelelő adjánlatokat és ki tudjuk exportálni csv fájlba.

Fő funkciói 
1. XML fájl beolvasás 
2. Adat tárolása osztály segítségével.
3. Szűrési lehetőségek. 
4. Adatok exportálása.
